How It Works

From idea to deployed code

Four steps to autonomous project completion.

1

Describe your project

Tell the PM agent what to build. It creates goals, tasks, and assigns roles.

2

Review and approve

Adjust the plan if needed. Click "Kickoff" to spawn worker agents.

3

Agents work autonomously

Each agent works in their own branch, commits code, and coordinates through shared context.

4

Merge and deploy

Review, merge to main, deploy. Done.

Get Started

Installation

No build step required. Clone, configure, run.

Terminal 
# Clone and install
git clone https://github.com/Adrian-LuMed/helix.git
cd helix && npm install

# Configure gateway connection
cp config.example.json config.json
# Edit config.json: set your OpenClaw gateway URL

# Start
npm start
