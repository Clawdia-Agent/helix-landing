How It Works

From idea to deployed code in autonomous steps.

1

Describe your project

Tell the PM agent what you want to build. It creates a plan with goals, tasks, and role assignments.

2

Review and approve

Check the plan. Adjust tasks if needed. Click "Kickoff" to spawn worker agents for each task.

3

Agents work autonomously

Each agent works in their own git branch. They update task status, commit code, and report blockers.

4

Merge and deploy

Review completed work. Merge branches. Deploy to Vercel, your server, or wherever you need.

Quick Start

Up and running in 2 minutes.

# Clone and install
git clone https://github.com/Adrian-Luerssen/Helix.git
cd clawcondos && npm install

# Configure your gateway connection
cp config.example.json config.json
# Edit config.json with your OpenClaw gateway URL

# Start the dashboard
node serve.js

# Open http://localhost:9000
